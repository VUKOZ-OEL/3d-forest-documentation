3D Forest
Software for analysis of Lidar data from forest environment.
Adding Third Party Libraries

Third party libraries used by 3D Forest are copied to '3rdparty' directory and therefore distributed together with 3D Forest.

3rd party code is build as a part of 3D Forest build process. This removes inconvenient step which would require to search, download and install dependencies.
